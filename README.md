# docker-desktop

> A docker image with some desktop applications, built from `masm/archlinux`

It currently includes:
* chromium
* firefox
* hipchat
* opera
* skype

## Install

Pull this image with `docker pull masm/desktop`.

## Usage

Run it with `docker run masm/glusterfs <app> <argument> ...`.
